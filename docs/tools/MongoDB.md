# MongoDB
## 数据库分类
数据库主要分成两种：
- 关系型数据库
  * MySQL、Oracle...
  * 关系型数据库中全都是表
- 非关系型数据库
  * MongoDB、Readis...
  * 键值对数据库
  * 文档数据库MongoDB
